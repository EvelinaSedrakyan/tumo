# Mad Libs Game 

A simple Python Mad Libs game where the user chooses a story template
and provides words to generate a funny story.
## Features
- 3 different story templates
- Input validation (no empty input, numbers checked)
- User-friendly prompts
## How to Run
1. Make sure Python 3 is installed
2. Run the file:
   python mad_libs.py
## Example
Choose a template and follow the prompts to create a custom story.
